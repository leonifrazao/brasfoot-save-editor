## Key Decisions
- **D-01:** Custom Bulk Objects (arrays of typed items).
- **D-02:** Typed endpoints (POST `/batch/players`, `/batch/teams`).
- **D-03:** Hard limit on batch sizes (synchronous 400 Bad Request, e.g. configurable `brasfoot.api.batch.max-size=1000`).
- **D-04:** Edits are Absolute Only (no deltas).
- **D-05:** Partial Success (207 Multi-Status response).
- **D-06:** Errors map back to request indices.
- **D-09:** Summary logging only.
- **D-10:** Realistic example payloads in Swagger.
</context>
